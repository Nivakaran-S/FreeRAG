# FAQ Documents

Place your FAQ documents here. These will be automatically loaded when the application starts.

## Supported Formats
- PDF (.pdf)
- Word documents (.docx)
- Text files (.txt)
- Markdown files (.md)

## Example Format
Create documents with Q&A pairs like:

```
Q: What is photosynthesis?
A: Photosynthesis is the process by which plants convert sunlight into energy...

Q: What are the benefits of exercise?
A: Regular exercise provides numerous health benefits including...
```

The system will automatically:
1. Ingest these documents at startup
2. Use them to answer user questions
3. Return instant responses for matching queries
